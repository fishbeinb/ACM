Sequencer following Conway's Game of Life 
Written for processing. Sound must be set up using this Pure Data https://github.com/scowalt/SIGMusic-processing-pd-example/blob/master/pd.pd.

CGOL - Sequence which follows CGOL's officiall rules.
CGOL_2LIVES - Simple modification to the rules. Each square now has 2 Lives. Much better for interesting visualization.
CGOL_KS - Many new features added, CGOL_2LIVES rules, interface should be cleaned up or labled. (KS stands for Kitchen Sink)
